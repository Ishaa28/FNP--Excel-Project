This Excel dashboard provides a comprehensive analysis of FNP sales data, highlighting key metrics such as total orders, revenue, average customer spend, and delivery time. 
The dashboard visualizes sales performance by occasion, category, month, and hour, offering insights into peak sales periods and top-performing products. 
Geographic distribution is also displayed, showing order concentration in major cities like Bangalore, Delhi, and Gurugram. 
Interactive elements allow users to filter and explore data for actionable business insights, making it a valuable tool for strategic decision-making and performance tracking.
